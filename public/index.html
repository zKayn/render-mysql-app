Thông tin Cloud Computing

Ứng dụng này được triển khai trên nền tảng Cloud với các tính năng:

  • Auto Scaling: Tự động điều chỉnh số lượng instances dựa trên tải
  • Data Replication: Sao chép dữ liệu qua nhiều vùng địa lý
  • Load Balancing: Phân phối request đến các instances khác nhau
  • Cloud Monitoring: Giám sát hiệu suất ứng dụng

Truy cập các tab trên thanh điều hướng để xem trực quan các tính năng.
